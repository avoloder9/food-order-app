# 🍔 Food Order App

A simple full-stack web app for browsing meals, adding them to a cart, and placing orders.  

## ✨ Features
- 🥗 Browse and select meals  
- 🛒 Add/remove items from the cart  
- 📝 Checkout with validation  
- 💾 Save orders on the server  
- 📧 Email confirmation  

## 🛠 Tech Stack
- React  
- Node.js  

## 🚀 Setup
```bash
# Frontend
npm install
npm run dev

# Backend
cd backend
npm install
node app.js
